ChatPlus Realm Editor

This is the realm editor for ChatPlus.

To create a new Realm, select the Add radio button, enter the realm id (2-8 alphanumeric characters) in the text box, and press "Make Change". If you wish To delete a realm, select the Delete radio button, enter the realm id you wish to delete in the text box, and press "Make Change".

There is no maximum to the number of realms you can create
